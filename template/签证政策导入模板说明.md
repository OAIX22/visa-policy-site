# Excel / 表格导入模板说明

支持 **`.xlsx`**（推荐）或 **`.csv`**（UTF-8）。首行必须为表头，列名需与下表字段名一致（**不区分大小写**）。

## 列定义

| 列名（表头） | 必填 | 内容说明 |
|--------------|------|----------|
| `country_code` | **是** | ISO 3166-1 **二位**国家/地区代码，大写，如 `TH`、`JP`、`US`。港澳台等特殊口径请与产品统一后再录入。 |
| `name_zh` | **是** | 国家或地区**中文常用名**，如 `泰国`、`日本`。 |
| `name_en` | 否 | **英文常用名**，如 `Thailand`，用于英文搜索。 |
| `aliases` | 否 | **别名与拼音**，多个用英文逗号 `,` 分隔，如 `泰,泰王国,taiguo,tg`。用户输入会与此列及中英文名一起做模糊匹配。 |
| `cities_zh` | 否 | **主要城市**，多个用英文逗号分隔；**建议中英文都写**（如 `曼谷,Bangkok`），用于「输入城市→关联到国家」的检索。 |
| `policy_type` | 否 | **政策类型标签**，建议从下列择一或组合（纯文本即可）：`免签`、`落地签`、`电子签`、`需事先办签`、`过境免签`、`其他`。 |
| `policy_summary` | **是** | **展示给用户的核心说明**（停留天数、入境条件、是否需返程票等）。Excel 单元格内换行：Windows 为 `Alt + Enter`。 |
| `official_url` | 否 | **官方信息链接**（使馆、移民局、外交部领事司等），完整 URL，以 `https://` 开头。 |
| `last_verified` | 否 | **人工核验日期**，格式 `YYYY-MM-DD`，如 `2026-05-01`。 |

## 填写注意

1. **编码**：CSV 请用 **UTF-8**（Excel 另存为时选择「CSV UTF-8」）。  
2. **合并规则**：导入时以 `country_code` 为唯一键。**同代码会覆盖站内该国的内置展示**，并覆盖本机此前已导入的相同代码（以最后一次导入为准）。  
3. **法律性质**：模板内填写内容仍属信息整理，**不构成签证法律意见**；请在 `official_url` 尽量指向权威来源。  
4. **空列**：未使用的列可留空，但表头必须齐全（或与下载模板一致）。

## 模板文件

- 同目录下的 `visa-policy-template.csv` 可用 Excel 打开后另存为 `.xlsx` 再填写。  
- 网站内提供「下载模板」按钮，格式与之一致。
